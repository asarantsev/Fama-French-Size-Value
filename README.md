# Fama-French-Size-Value
Consider size and value returns. We have autoregression fit with and without volatility. For the value, we are successful, but for the size, we failed. verification.py is the auxillary function which studies regression residuals. factors.xlsx and factors.py are the main data and code files. They are taken from 

https://mba.tuck.dartmouth.edu/pages/faculty/ken.french/ftp/F-F_Research_Data_Factors_CSV.zip 

with 1927-2025 annual data. The blog post is 

https://my-finance.org/2026/05/25/fama-french-factors-time-series-modeling/
